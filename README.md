# Okene Tech Hub Premium Website

This package contains a complete multi-page frontend website for Okene Tech Hub.

## Included
- Home page
- About page
- Programs page
- Events page
- Stories page
- Contact page
- Shared CSS and JS assets

## Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Poppins + Inter)

## Accessibility notes
- Semantic landmarks and headings
- Skip link
- Visible focus states
- Reduced motion support via `prefers-reduced-motion`
- Keyboard-friendly navigation
- High-contrast palette
- Form labels and clear input states

## Notes
- The contact form is frontend-only and should be connected to a backend or form service.
- Partner logos, real testimonials, phone number, and event dates are placeholders and should be updated with verified content.
- This build was created from the conversation requirements and the brand direction already discussed. I could not reliably re-read the uploaded PDFs because the upload tool repeatedly expired them during the session.

## Local preview
Open `index.html` directly in a browser, or serve the folder with any static server.
